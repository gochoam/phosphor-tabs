Class TabBar

A widget which displays a row of tabs.

Notes

A TabBar widget does not support child widgets. Adding children to a TabBar will result in undefined behavior.

A property descriptor which controls the hidden state of a widget.

Notes

This property controls whether a widget is explicitly hidden.

Hiding a widget will cause the widget and all of its descendants to become not-visible.

This property will toggle the presence of [[HIDDEN_CLASS]] on a widget according to the property value. It will also dispatch 'after-show' and 'before-hide' messages as appropriate.

attachTab

  • attachTab(tab: Tab, clientX: number): boolean
    • Defined in tabbar.ts:546

    Add a tab to the tab bar at the given client X position.

    Parameters

    • tab: Tab

      The tab to attach to the tab bar.

    • clientX: number

      The current client X mouse position.

    Returns boolean

    true if the tab was attached, false otherwise.

    Notes

    This method is intended for use by code which supports tear-off tab interfaces. It will insert the tab at the specified location and grab the mouse to continue the tab drag. It assumes that the left mouse button is currently pressed.

    This is a no-op if the tab is already contained by the tab bar, if the tabs are not movable, or if a tab drag is in progress.
