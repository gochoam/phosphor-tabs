Class TabPanel

A panel which provides a tabbed layout for child widgets.

The TabPanel provides a convenient combination of a TabBar and a StackedPanel which allows the user to toggle between widgets by selecting the tab associated with a widget.

Notes

Widgets should be added to a TabPanel using the <prefix>Widget methods, not the <prefix>Child methods. The children of a TabPanel should not be manipulated directly by user code.
